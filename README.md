# Back-end Challenge

### Tecnologias Utilizadas

Linguagem: TypeScript
Framework: Node.js
Banco de Dados: SQLite

## Instalação e Uso

1. Clone este repositório:

```bash
git clone https://github.com/murilocb/Intera-test-backend.git
```

```bash
cd Intera-test-backend
```

Instale as dependências:

```bash
npm install
```

Rodando o projeto:

```bash
npm run dev
```
